## One Command Generation

```bash
make build-sdk
```

**What it does:**
1. Validates all API specifications
2. Generates WebSocket models and handlers
3. Generates HTTP clients with OpenAPI Generator
4. Fixes all imports automatically
5. Creates unified client wrapper
6. Validates the complete SDK

**Time:** ~2-3 minutes
**Result:** Complete, working SDK ready to use

## Requirements

- Python 3.8+
- Java 11+ (for OpenAPI Generator)
- Dependencies: `pip install -r requirements.txt`

## Usage After Generation

```python
from whitebit_sdk import WhiteBitClient

# Initialize
client = WhiteBitClient(api_key="...", api_secret="...")

# Use APIs
await client.public_api.public.api_v4_public_markets_get()
await client.main_api.main_account.get_main_balance(...)
```
